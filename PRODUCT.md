# Product

## Register
product

## Users
Small marketing agencies (or solo media buyers), likely based in India (using INR/₹), managing lead-gen campaigns for multiple clients (real estate, coaching, local services, B2B).
They check the dashboard daily (often on mobile between client calls) to monitor performance across multiple projects simultaneously.

## Product Purpose
To quickly highlight which client campaigns are wasting spend and which ones are performing well. It enables the user to catch inefficiencies fast, prioritize which clients need attention today, and share automated WhatsApp weekly reports. Success looks like saving time on daily performance checks and minimizing wasted ad spend.

## Brand Personality
Clean, fast, trustworthy. Vibe is similar to Vercel (neutral slate/white, restrained color, monospace-adjacent precision) and Linear (tight spacing, subtle shadows, no clutter).

## Anti-references
- Flashy/gamified growth-hacker dashboards with big colorful donut charts and badges.
- Pretty but slow, illustration-heavy landing-page style dashboards.
- Clunky enterprise tools from 2010.
- Forced dark mode as a default.

## Design Principles
- **Data over decoration:** The UI should optimize for fast scanning and comparison of tabular data, not admiring visuals.
- **Urgency and clarity:** Instantly highlight what needs attention today without overwhelming the user with noise.
- **Precision and trust:** Use restraint in typography and spacing to evoke competence and reliability.

## Accessibility & Inclusion
High contrast for fast readability on the go (potentially outdoors on mobile). Avoid relying purely on color (like red/green) to indicate performance; ensure layout and typography carry the hierarchy.
